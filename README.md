# Vigiles Aeternae — Corpus Mythicum

**The Library of All Libraries** — a research corpus synthesizing mythology across 20+ traditions into formal governance philosophies, comparative power taxonomies, and the metaLAWs codex.

## What This Is

No regime enters the Agon without its philosophy being catalogued here first. This is the Archive of the Vigiles — where every mythology is studied, every power structure mapped, every governance model formalized.

## Structure

```
corpus/          Per-tradition mythology syntheses
  greek-roman/   Titans, Olympians, Furies, mystery cults
  tolkien/       Delegated authority, Ainur hierarchy
  malazan/       Pressure ascendancy, Deck of Dragons
  matrix/        Program hierarchy, Agent Smith protocols
  marvel-cosmic/ Celestials, Living Tribunal, Watchers
  ...            27 tradition directories
regimes/         Formal regime YAML specifications
taxonomy/        Cross-mythology power taxonomy schemas
metalaws/        The metaLAWs codex (100+ universal laws across 9 domains)
sources/         Bibliography and source references
```
